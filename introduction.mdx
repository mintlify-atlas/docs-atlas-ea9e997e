# TelecomX LATAM — Customer Churn Analysis

This project delivers a complete data analysis pipeline for understanding and reducing customer churn at **TelecomX**, a fictional Latin American telecommunications company. By combining ETL processing with Exploratory Data Analysis (EDA), it uncovers the key drivers behind customer attrition and provides strategic recommendations to reduce the 26.5% churn rate.

## Key findings at a glance

| Metric | Value |
|--------|-------|
| Total customers analyzed | 7,267 |
| Overall churn rate | **26.5%** |
| Highest-risk group | Month-to-month contracts |
| Most vulnerable period | First 6 months of tenure |
| Churn-correlated factor | Absence of tech support subscription |
