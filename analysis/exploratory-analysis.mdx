## What is EDA?

Exploratory Data Analysis (EDA) is the initial step in any data science project. It involves summarizing the main characteristics of a dataset — often using statistical measures and visualizations — before applying any formal modeling techniques.

In the context of customer churn, EDA serves to:

- Understand the shape and quality of the dataset
- Identify missing values, outliers, and inconsistencies
- Reveal the distribution of the target variable (`Churn`)
- Discover relationships between features and churn behavior

<Note>
The TelecomX dataset was loaded from a nested JSON file (`TelecomX_Data.json`). The raw structure contained 7,267 rows and 6 columns, where each column held nested dictionaries that required flattening before analysis.
</Note>

## Loading and Initial Inspection

The notebook uses a custom loading function that reads the raw JSON and inspects the DataFrame structure:

```python
import pandas as pd
import json

file_path = 'TelecomX_Data.json'

def cargar_datos_locales(ruta):
    try:
        with open(ruta, 'r', encoding='utf-8') as f:
            datos = json.load(f)

        df = pd.DataFrame(datos)

        print(" Datos cargados correctamente desde el archivo local.")
        print(f"Dimensiones: {df.shape[0]} filas y {df.shape[1]} columnas.")
        return df

    except FileNotFoundError:
        print(f" Error: El archivo '{ruta}' no se encuentra en el entorno actual.")
        return None
    except Exception as e:
        print(f" Ocurrió un error inesperado: {e}")
        return None

# Ejecución
df_telecom = cargar_datos_locales(file_path)

if df_telecom is not None:
    display(df_telecom.head())
    print("\nTipos de datos detectados:")
    print(df_telecom.info())
```

## Feature Engineering: Daily Charges

A new feature `Cuentas_Diarias` (daily charge) was engineered by dividing the monthly charge by 30:

```python
# Creamos la columna Cuentas_Diarias
df_limpio['Cuentas_Diarias'] = (df_limpio['Charges.Monthly'] / 30).round(2)

# Visualización de la distribución de gastos diarios
import matplotlib.pyplot as plt
import seaborn as sns

plt.figure(figsize=(10, 5))
sns.histplot(df_limpio['Cuentas_Diarias'], kde=True, color='teal')
plt.title('Distribución del Gasto Diario por Cliente', fontsize=14, fontweight='bold')
plt.xlabel('Costo Diario (MXN/USD)')
plt.ylabel('Frecuencia de Clientes')
plt.show()
```

This metric provides finer granularity for cost-based churn analysis and is used throughout the numerical variable analysis.
