The extraction step reads `TelecomX_Data.json` from disk and loads it into a pandas DataFrame using a
defensive helper function that handles missing files and unexpected runtime errors gracefully.

<Note>
  The raw file contains **7,267 rows** and **6 columns**. Four of those columns (`customer`, `phone`,
  `internet`, and `account`) hold nested JSON objects that are flattened in the subsequent
  [Transformation](/pipeline/transformation) step.
</Note>

## Raw data structure

After loading, `df_telecom` has the following top-level columns:

| Column | dtype | Description |
|---|---|---|
| `customerID` | object | Unique customer identifier (e.g. `0002-ORFBO`) |
| `Churn` | object | Target label — `"Yes"`, `"No"`, or empty string (`""`) |
| `customer` | object | Nested dict: demographics (gender, SeniorCitizen, Partner, Dependents, tenure) |
| `phone` | object | Nested dict: phone services (PhoneService, MultipleLines) |
| `internet` | object | Nested dict: internet services (InternetService, OnlineSecurity, OnlineBackup, DeviceProtection, TechSupport, StreamingTV, StreamingMovies) |
| `account` | object | Nested dict: account details (Contract, PaperlessBilling, PaymentMethod, Charges.Monthly, Charges.Total) |

## The `cargar_datos_locales()` function

```python title="TelecomX_LATAM.ipynb — Extracción"
import pandas as pd
import json


file_path = 'TelecomX_Data.json'

def cargar_datos_locales(ruta):
    try:
        # Abrimos el archivo local
        with open(ruta, 'r', encoding='utf-8') as f:
            datos = json.load(f)

        # Convertimos a DataFrame
        df = pd.DataFrame(datos)

        print(" Datos cargados correctamente desde el archivo local.")
        print(f"Dimensiones: {df.shape[0]} filas y {df.shape[1]} columnas.")
        return df

    except FileNotFoundError:
        print(f" Error: El archivo '{ruta}' no se encuentra en el entorno actual.")
        return None
    except Exception as e:
        print(f" Ocurrió un error inesperado: {e}")
        return None

# Ejecución
df_telecom = cargar_datos_locales(file_path)

# Verificación inicial de la estructura de Telecom X
if df_telecom is not None:
    display(df_telecom.head())
    print("\nTipos de datos detectados:")
    print(df_telecom.info())
```

## Churn label distribution in raw data

The raw `Churn` column contains **three** distinct values, not two:

| Value | Count | Percentage |
|---|---|---|
| `"No"` | 5,174 | ~71.2 % |
| `"Yes"` | 1,869 | ~25.7 % |
| `""` (empty string) | 224 | ~3.1 % |

<Warning>
  The ~3% of rows with an empty `Churn` string represent records with **missing target labels**.
  These are dropped during the [Transformation](/pipeline/transformation) step, reducing the
  dataset from 7,267 to **7,043 rows**.
</Warning>
